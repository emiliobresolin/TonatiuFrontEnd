Descrição do Serviço

Correção/Instalação de Disjuntores

Instalação:
1-Avaliação do sistema elétrico: Antes da instalação do disjuntor, é necessário avaliar o sistema elétrico para determinar a capacidade de carga adequada e a localização apropriada para o disjuntor.
2-Escolha do disjuntor correto: Com base na avaliação do sistema elétrico, é selecionado um disjuntor com a capacidade de corrente nominal apropriada para a carga que ele protegerá.
3-Preparação do painel elétrico: O painel elétrico onde o disjuntor será instalado precisa estar adequadamente preparado.
4-Teste e verificação: Após a instalação, é importante realizar testes para garantir o correto funcionamento do disjuntor.
Correção:
1-Inspeção visual: Periodicamente, os disjuntores devem ser inspecionados visualmente para verificar se há danos físicos.
2-Verificação das conexões: É importante verificar se as conexões elétricas do disjuntor estejam apertadas e seguras.
3-Testes de funcionamento: Periodicamente, os disjuntores devem ser testados para verificar se estão funcionando corretamente.

1-Avaliação do sistema elétrico existente: Antes de realizar a troca da fiação, é necessário avaliar o sistema elétrico atual.
